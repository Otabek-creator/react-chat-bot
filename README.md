# Simple Chatbot

A simple web-based chatbot built as a **learning project** using HTML, CSS, JavaScript, and React.

This project was created while following free tutorials by **SuperSimpleDev** and is intended for **educational purposes only**.

---

## Features
- Chat-style user interface
- User and bot messages
- Simple predefined chatbot responses
- Runs entirely in the browser (no backend)
